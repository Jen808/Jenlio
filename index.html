My-Widdy
Desktop widget app for clock, calendar, todo-list, and weather. Each function has more than 4-6 designs with 16 different color themes. Users can connect Google account to fetch events and schedule. Installation file is available. Codes are available on Github.

Vendors
This project makes it easier for street vendors and customers to connect, especially during COVID-19 when finding vendors became more difficult. The goal is to help customers find vendors from home and help sellers boost their sales.
IFE System
This project aims to enrich the in-flight entertainment (IFE) system thereby elevating the overall travel experience. Drawing upon feedback from customers of various airlines, this initiative seeks to address their needs and concerns, minimizing inconvenience while maximizing convenience
