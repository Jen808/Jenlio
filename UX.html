UX Research and Design

In Flight Entertainment System

This project is about improving the travel experience, targeting increased convenience and customer attraction to flying based on user feedback.

FRESH

This initiative explores consumer interest in buying food directly from producers, aiming to assess demand, willingness, and the feasibility of a direct farm-to-consumer sales system

VENDORS

This project aims to connect street vendors with customers via an app, overcoming COVID-19 challenges by facilitating product sales and making it easier for customers to find vendors, addressing needs like location, dates, and pricing, despite obstacles like vendor decline and illegal vending.

GLANSIS

This project aims to redesign the GLANSIS initiative by NOAA, focusing on improving the Educator Hub based on insights from Rochelle Sturtevant, the program manager. It involves managing extensive datasets and understanding specific organizational needs.

RoosRoast

This project entails the professional redesign of the RoosRoast Coffee website, a local café renowned for its coffee, gear, coffee-making tools, and vibrant community engagement.

AmongUs

This project aims to identify and address user inconveniences in the Among Us game by redesigning its interface and functionality to enhance user experience..
